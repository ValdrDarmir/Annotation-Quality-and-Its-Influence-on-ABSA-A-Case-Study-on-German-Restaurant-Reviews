Willkommen zur Annotationsaufgabe

Aspekt-basierte Sentiment Analyse ist eine Methode zur Untersuchung und Bewertung von Meinungen und Gefühlen in Texten, die sich auf spezifische Aspekte oder Merkmale eines Produkts, einer Dienstleistung oder eines Themas konzentriert.

Dazu führen Sie folgende Schritte aus:
1. Sie wählen dafür aus einer Liste die zutreffenden Kombinationen (Tupel) von Kategorie und Polarität aus - zum Beispiel „Service–Negativ“ oder „Essen–Positiv“. Es können auch mehrere Tupel pro Text ausgewählt werden, wenn unterschiedliche Aspekte angesprochen werden.

2. Bitte nutzen Sie die Metatags, wenn:
- die Annotation schwierig ist,
- wichtige Informationen zur Interpretation fehlen,
Zusätzliche Hinweise oder Probleme können Sie im Kommentarfeld dokumentieren.
